En este caso he optado por utilizar React con typescript, en el ambito del dise√±o he utilizado la libreria NextUI, quien integra tailwindcss, en mi opinion es muy poderosa y agradable a la vista. ## Entre las librerias utilizadas esta:
1. React-router-DOM
2. React-hook-Form
3. React Query
4. Redux-Toolkit
5. NextUI
6. Tailwindcss

## Guia de instalacion
1. Clonar el proyecto
2. Renombrar el archivo ```.env.local.example``` por ```.env.local```
3. instalar dependencias ```yarn install```
4. Ejecutar la aplicacion ```yarn dev```
